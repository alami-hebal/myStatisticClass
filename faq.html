Basic Concepts Linear Equations Matrices & Vectors

What is a linear equation?

A linear equation is an algebraic expression in which each term is either a constant or the product of a constant and a single variable, and the graph of the equation forms a straight line.

What is the difference between a variable and a constant?

A variable is a symbol that represents an unknown value, while a constant is a fixed, known value in an equation or expression.

What does it mean to solve an equation?

Solving an equation means finding the value(s) of the variable(s) that make the equation true.

What is the Cartesian coordinate system?

The Cartesian coordinate system uses two axes (x and y) to define points on a plane, commonly used to graph linear equations.

What is the slope of a line?

The slope is a measure of how steep a line is and is calculated as the ratio of the vertical change to the horizontal change between two points on the line.

How do I solve a one-variable linear equation?

You solve by isolating the variable on one side of the equation using inverse operations such as addition, subtraction, multiplication, and division.

What is a system of linear equations?

A system of linear equations consists of two or more linear equations with the same variables. The solution is the point(s) where the equations intersect.

What are the methods to solve a system of equations?

Common methods include graphing, substitution, and elimination. Matrix methods such as Gaussian elimination can also be used.

How do I interpret the solution of a system?

A system can have one solution (intersecting lines), no solution (parallel lines), or infinitely many solutions (same line).

How is graphing used to solve linear equations?

By plotting each equation on a graph, the point(s) where the lines intersect represent the solution(s) to the system.

What is a matrix?

A matrix is a rectangular array of numbers arranged in rows and columns, often used to represent systems of linear equations.

How are matrices used to solve equations?

Matrices can be used to represent and solve systems of equations through methods like matrix inversion and row reduction.

What is a vector in linear algebra?

A vector is an ordered set of numbers representing a point or direction in space. Vectors are essential in describing linear transformations.

What is a linear transformation?

A linear transformation is a function that maps vectors to vectors in a way that preserves vector addition and scalar multiplication, often represented by matrices.

What is the determinant of a matrix and why is it important?

The determinant is a scalar value that indicates whether a matrix is invertible. A non-zero determinant means the matrix has an inverse and the system has a unique solution.
